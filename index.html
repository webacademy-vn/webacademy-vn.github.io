Frontend

На даний момент у нас єдиний курс Frontend (HTML, CSS, JavaScript).
Ми вивчаємо такі популярні технології як адаптивний дизайн з Bootstrap, препроцесор CSS - SASS, Angular останньої версії, Angular Material, NGRX, GIT, TypeScript, OOP.
